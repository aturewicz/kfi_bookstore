# kfi_bookstore - frontend

## Tech
Project uses:
* Vue.js ver. 3.0.0 
  
If you want see all dependencies check files:
* package.json

## Project setup
```
yarn install
```

## Compiles and hot-reloads for development
```
yarn serve
```

## Compiles and minifies for production
```
yarn build
```
